⚠️ Tryb lokalny (file://) — wymagany eksport

Strona działa lokalnie z dysku. W tym trybie admin.html i index.html nie dzielą localStorage — zmiany nie trafią automatycznie na stronę.

Jak to naprawić:
1. Po zmianach kliknij „Eksportuj dane" poniżej
2. Zapisz plik gryf_data.json w tym samym folderze co index.html
3. Strona główna automatycznie go wczyta

Docelowo wgraj oba pliki na hosting — wtedy synchronizacja działa sama.

💾 Eksport / Import danych

Eksportuj dane kalendarza do pliku JSON — przenieś je na inne urządzenie lub wykonaj kopię zapasową. Możesz też zaimportować wcześniej zapisany plik.
